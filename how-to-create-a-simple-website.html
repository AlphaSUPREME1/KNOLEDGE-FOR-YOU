How to Create a Simple Website Using HTML and CSS

Creating a website is one of the best skills a student can learn today. In this tutorial, you will learn how to create a simple website using HTML and CSS step by step.

What You Need Before Starting

  • Basic computer knowledge
  • A text editor (VS Code or Notepad)
  • A web browser

Step 1: Create the HTML File

HTML is used to create the structure of a website. Create a file named index.html and add the following code: 


<!DOCTYPE html>
<html>
<head>
    <title>My First Website</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is my first website.</p>
</body>
</html>

Step 2: Add CSS for Design

CSS is used to make the website look attractive. Create a file named style.css and add: 


body {
    font-family: Arial, sans-serif;
    background-color: #f1f5f9;
    text-align: center;
}

h1 {
    color: #2563eb;
}

Step 3: Connect CSS to HTML

Link the CSS file inside the HTML <head> section: 


<link rel="stylesheet" href="style.css">

Why Learning Web Development is Important

Web development skills can help students earn money online, create personal projects, and understand how the internet works.

Conclusion

You have now learned how to create a simple website using HTML and CSS. Practice more, improve your design, and keep learning.
